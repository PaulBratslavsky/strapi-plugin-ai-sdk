# 3D Avatar Models

Place `.glb` or `.gltf` model files here.

## Recommended Sources (Free, CC-BY or CC0)

1. **Awa Meiko** (CC-BY) — Anime girl with 163 bones + 57 blendshapes
   https://sketchfab.com/3d-models/awa-meiko-anime-girl-refixed-glb-filepbr-59d35de0ab2849b6b8e63e8171c2d14e

2. **Chibi Base Mesh** (CC-BY) — Cute low-poly chibi, rigged
   https://sketchfab.com/3d-models/free-pack-chibi-base-mesh-rigged-fed4fb329f224f1594f631eae8d2626b

3. **VRoid CC0 Models** (CC0) — No attribution, VRM/glTF based
   https://opengameart.org/content/vroid-studio-cc0-models

## Usage

1. Download a `.glb` file and place it here (e.g. `avatar.glb`)
2. In `Avatar3D.tsx`, change `MODEL_PATH` to point to it
3. Rebuild the plugin: `npx strapi-plugin build`
